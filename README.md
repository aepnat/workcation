2. `yarn build` : Build files

### Development

Please using `yarn serve` for development workflow

### Packages

- Vue - Javascript Framework
- TailwindCSS - CSS Framework
- PostCSS - Transforming CSS with Javascript
- PurgeCSS - Remove unused CSS
